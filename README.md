# LawReady

로스쿨 지원자의 GPA, LEET, 영어 성적, 대학 및 정성 요소를 바탕으로 전국 25개 로스쿨의 상대적 합격 가능성을 보여주는 교육용 프로토타입입니다.

## 로컬 실행
정적 파일이므로 `index.html`을 열거나 아래 명령을 실행합니다.

```bash
python3 -m http.server 8000
```

## GitHub Pages 배포
1. 새 GitHub 저장소를 만듭니다.
2. 이 폴더의 파일을 저장소 루트에 업로드합니다.
3. 저장소 `Settings > Pages`에서 `Deploy from a branch`를 선택합니다.
4. `main` 브랜치와 `/ (root)`를 선택하고 저장합니다.

## 주의
본 서비스의 계산식과 결과는 실제 입시 데이터에 기반한 공식 예측이 아닌 데모용 휴리스틱입니다. 실제 지원 판단에는 각 로스쿨의 최신 모집요강과 공식 입시 결과를 함께 확인해야 합니다.
